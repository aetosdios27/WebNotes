# WebNotes CLI

Rust CLI for local note workflows used by the WebNotes project.

## Prerequisites

- Rust toolchain (`rustup`, `cargo`)

## Common Commands

- Run: `cargo run`
- Test: `cargo test`
- Lint: `cargo clippy --all-targets --all-features`
- Format: `cargo fmt`

## Notes

- Build artifacts are generated under `cli/target/` and are intentionally not tracked by git.
